# KIP Knowledge Base — Obsidian vault

This folder **is** an Obsidian vault. Open it directly in Obsidian
(`Open folder as vault` → select this `vault/` folder).

- Start at [[KIPs]] — the index / map of content.
- Each note under `KIPs/` is one Kafka Improvement Proposal, with metadata in
  the frontmatter (status, category, tags, related) and the full write-up in the
  body (Motivation, Design, Trade-offs, Rejected Alternatives, Discussion, Vote).
- `related` links are real `[[wikilinks]]`, so the graph view and backlinks work.

These notes are the **single source of truth**. The web viewer in `../viewer`
parses them at build time — edit the markdown here and the viewer follows.
